# EchoMonitor 开发计划

## 项目概述

**应用名称**: EchoMonitor（环境音实时监听）
**核心功能**: 实时麦克风收音 → 即时播放，最小化延迟
**技术难点**: 低延迟音频处理、后台保活、系统级音频焦点管理

## 已完成阶段

### Phase 1: 基础架构与权限 ✅
- Android项目初始化
- 权限管理系统 (RECORD_AUDIO, FOREGROUND_SERVICE等)
- 前台服务框架
- 基础UI界面

### Phase 2: 低延迟音频实现 ✅
- Oboe音频库集成
- C++音频引擎
- 环形缓冲区管理
- JNI桥接
- 目标延迟: < 20ms

### Phase 3: 音量控制系统 ✅
- VolumeController音量管理
- 软限幅算法
- 最大音量限制(dB转线性)
- 防止音频削波

### Phase 4: 后台保活机制 ✅
- 前台服务通知
- WakeLock防止CPU休眠
- 电池优化白名单
- 开机自启动支持

### Phase 5: 系统级音频覆盖 ✅
- 音频焦点管理
- USAGE_ASSISTANCE_SONIFICATION
- 音频路由策略
- 强制可听配置

## 文件清单

### Kotlin源码
- MainActivity.kt - 主界面
- EchoApplication.kt - Application类
- AudioMonitoringService.kt - 前台服务
- PermissionManager.kt - 权限管理
- AudioEngine.kt - 音频引擎桥接
- VolumeController.kt - 音量控制

### C++源码
- audio_engine.h/.cpp - Oboe音频引擎
- volume_limiter.h/.cpp - 音量限制器
- jni_bridge.cpp - JNI桥接
- CMakeLists.txt - 构建配置

### 资源文件
- AndroidManifest.xml - 权限声明
- activity_main.xml - 主界面布局
- strings.xml - 字符串资源
- themes.xml - 主题配置

## 构建说明

### 环境要求
- Android Studio Hedgehog+
- NDK 25.1.8937393
- CMake 3.22.1
- JDK 17

### 构建命令
```bash
./gradlew assembleDebug
./gradlew installDebug
```
