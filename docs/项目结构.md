## 核心类说明

### Kotlin层

| 类名 | 职责 | 对应Phase |
|------|------|-----------|
| MainActivity | 主界面、用户交互 | Phase 1/6 |
| AudioMonitoringService | 前台服务、音频控制 | Phase 1/4/5 |
| PermissionManager | 权限管理 | Phase 1 |
| AudioEngine | C++桥接、原生调用 | Phase 2 |
| VolumeController | 音量管理、持久化 | Phase 3 |

### C++层

| 文件 | 职责 | 对应Phase |
|------|------|-----------|
| audio_engine.cpp | Oboe音频流管理 | Phase 2 |
| volume_limiter.cpp | 软限幅算法 | Phase 3 |
| jni_bridge.cpp | JNI接口 | Phase 2 |

## 依赖关系图

```
MainActivity
    ├── PermissionManager (Phase 1)
    ├── VolumeController (Phase 3)
    └── AudioMonitoringService (Phase 1/4/5)
            ├── AudioEngine (Phase 2)
            │       └── JNI → audio_engine.cpp
            └── AudioManager (Phase 5)
                    └── 音频焦点管理
```
